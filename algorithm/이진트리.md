# 이진트리

- 자식 노드가 2개 이하인 트리
- 완전 이진 트리 : 노드를 왼쪽부터 삽입
  => 마지막 레벨을 제외한 나머지 레벨이 다 채워져 있음

- 포화 이진 트리 : 모든 리프 노드의 레벨이 동일하고, 노드가 존재하는 모든 레벨이 가득 채워져 있는 이진 트리

## 이진트리 순회

트리 내 어떤 자료가 담겨 있는지 알기 위해 탐색

- 이동 순서는 재귀적 성질 이용
- (전위) ROOT -> L -> R => 실제 방문 순서 알 수 있음
  - 맨 앞이 루트
- (중위) L -> Root -> R
  - 루트의 왼쪽, 오른쪽 자식 구분 가능
- (후위) L -> R -> Root
  - 맨 뒤가 루트.

단지 출력하는 시점이 다름
