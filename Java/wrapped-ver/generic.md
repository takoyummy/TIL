# Generic

- 제네릭을 사용하면 잘못된 타입이 들어올 수 있는 것을 컴파일 타임에 방지할 수 있음
- 클래스 외부에서 타입을 지정해주기 때문에, 타입을 체크하고 변환해줄 필요 없음. 관리하기 편해짐.
- 비슷한 기능을 지원하는 경우 코드의 재사용성이 높아짐.

- <T> : Type
- <?> : wild card, unknown type (알 수 없는 type)
- <E> : Element
- <K> : Key
- <V> : value
- <N> : Number
